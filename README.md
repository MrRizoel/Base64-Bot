<h1 align="centre"> Coder-Bot! </h1>

<b> Info </b>: Encode/Decode telegram bot! || Pyrogram based

----

<b> Hosting -! </b>

- `git clone https://github.com/MrRizoel/Coder-Bot`
- `cd Coder-Bot` 
- `pip3 install -r requirements.txt`
  Rename `example.env` as `.env` and fill variables!
- `python3 main.py`
